🚀 الخطوات الأولى:

1. تشغيل قاعدة البيانات:
   mysql -u root -p < database-schema.sql

2. تشغيل المشروع:
   npm run dev

3. الدخول:
   http://localhost:3000

4. لوحة التحكم:
   http://localhost:3000/admin/login
   المستخدم: admin
   كلمة المرور: 12345
