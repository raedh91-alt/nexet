## 🚨 إذا واجهت مشكلة

### ❌ "Error: Can't connect to MySQL"
**الحل**: شغّل خدمة MySQL
```bash
net start MySQL80
```

### ❌ "Port 3000 is already in use"
**الحل**: استخدم منفذ آخر
```bash
npm run dev -- -p 3001
```

### ❌ "Unknown database"
**الحل**: شغّل ملف البيانات
```bash
mysql -u root -p < database-schema.sql
```

### للمزيد من المشاكل
👉 اقرأ **SETUP_GUIDE.md** - فيه الحلول الكاملة
